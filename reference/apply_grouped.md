# Functional for Precomputed D4 Orbit Groups

Functional for Precomputed D4 Orbit Groups

## Usage

``` r
apply_grouped(spec_mat, groups, fun, ...)
```

## Arguments

- spec_mat:

  N x M matrix

- groups:

  List of indices (orbits)

- fun:

  A function that takes a vector and returns a vector of the same length

- ...:

  Additional arguments passed to fun
