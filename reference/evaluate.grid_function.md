# evaluate a grid function object

Evaluates a function on \\\[-\pi, \pi\]^2\\

## Usage

``` r
# S3 method for class 'grid_function'
evaluate(f, N = 30, M = 30)
```

## Arguments

- f:

  the grid_function object

- N:

  integer for number of rows to generate

- M:

  integer for number of columns to generate
